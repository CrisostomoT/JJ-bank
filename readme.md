# Como Usar
Clonar el repositorio localmente
- git clone git@github.com:Khadosh/JJ-bank.git
- cd JJ-bank
- npm install
- npm run start

## Estructura de Carpetas
- public: unicamente el HTML
- src: todo lo demas (javascript, imagenes, css)
